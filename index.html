Hello, I'm tirimid (sometimes Tirimid or polytopelover) and this is my personal website for publishing things I make - which means essays, guides, documentation, and so on. Whenever I create something that I want to put out into the world for others to see, I will probably create a page about it on this site.

I am strongly in favor of free software, privacy, and security. The source code to this website is available at its GitHub.

Leaderboard rules

  1. You must be using the base version of the Bavis Simulator you want to submit a score for, modified versions with only cosmetic changes but equivalent gameplay are permitted

  2. Strictly NO automation in the gameplay unless it is a feature of the base game, including keyboard macros, autotypers, or any other automation tool not included in Bavis Simulator

  3. Use a common keyboard layout (e.g. QWERTY, Dvorak, Colemak), not a custom layout designed to optimize input efficiency, when getting scores you plan to submit

Submitting scores

Once you get a score in Bavis Simulator worth submitting, you should send an email to bavis@tirimid.net containing the details of your score. The subject line of this email should be something along the lines of "Score Submission for Bavis Simulator Leaderboard". I will go over it and review it, and if it makes sense, I will add the score to the below leaderboard. If you are using a version of Bavis Simulator that does not have a leaderboard, you may request the creation of one by sending an email to bavis@tirimid.net with a subject line like "Request for Creation of New Bavis Simulator Leaderboard".

Please make sure to include the following in your score submission email:

  1. Which spin / version of Bavis Simulator you got your score on, as this will determine which leaderboard the score is registered on

  2. The details of the score (i.e. lives lost and bought, bavis count, multiplier, special bavises)

  3. Time taken to achieve the score (if your version of Bavis Simulator supports tracking this information)

  4. Evidence of legitimacy (not mandatory but I will request this before adding any especially suspicious scores to the leaderboard)

... and the following in your leaderboard request email:

  1. The name of the Bavis Simulator spin

  2. The download link with source code. Proprietary Bavis Simulators will not be put on the leaderboard

  3. Which fields the leaderboard should track in each score
